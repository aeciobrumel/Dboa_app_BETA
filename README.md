## Guia de instalacao

O projeto Expo fica em `rn-expo-app`.

### Pre-requisitos

- Node.js e npm
- Expo CLI (via `npx expo`)
- Android Studio e/ou Xcode (opcional, para emuladores)

### Instalacao e execucao

```bash
cd rn-expo-app
npm install
npm run start
```

Comandos uteis:

```bash
npm run android
npm run ios
npm run web
npm run test
npm run lint
npm run format
```
