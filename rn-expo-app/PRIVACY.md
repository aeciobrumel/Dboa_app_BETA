# Política de Privacidade

Este aplicativo armazena seus dados localmente no dispositivo. Não coletamos, compartilhamos nem transferimos informações pessoais para servidores.

Dados processados
- Cartões criados por você (título, texto e imagem opcional)
- Preferências do app (idioma, volumes, voz/velocidade/tom de TTS, posição da respiração guiada)

Onde ficam os dados
- No próprio dispositivo (AsyncStorage). Ao desinstalar o app, os dados são removidos.

Permissões
- Biblioteca de fotos: para você selecionar imagens e anexar aos cartões
- Acesso a arquivos (Android): para você escolher uma música de fundo
- Áudio em segundo plano: para música ambiente continuar tocando se a tela apagar

Tecnologias
- TTS do sistema (expo‑speech no Android/iOS, SpeechSynthesis no Web). Não enviamos o texto a serviços externos.

Contato
- Substitua por e‑mail/website oficial para suporte e solicitações de privacidade.
